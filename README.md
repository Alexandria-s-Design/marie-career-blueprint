# Marie Career Blueprint

**Dr. Marie Martin's Career Blueprint: AI, Equity & Workforce Development**

Professorship pathway modeled on Dr. Megan Bang — a scholar-practitioner who bridged community-based work with academic research.

## Mission

Transition Dr. Marie Martin from practitioner-leader to full-time faculty, targeting Professor of Practice, Clinical Faculty, or Program Coordinator roles in Education, Equity, and Workforce Development.

## Structure

```
career-arc/          - Career narrative and positioning documents
applications/        - Cover letters, statements per institution
publications/        - Journal articles, manuscripts, submission tracker
conferences/         - Conference submissions, presentations, CFP tracker
outreach/            - Networking emails, contact database
core-materials/      - CV, DEI statement, teaching philosophy, references
portfolio/           - Portfolio site source
pipeline/            - Position tracker, deadline calendar
.planning/           - Session state for multi-session work
```
